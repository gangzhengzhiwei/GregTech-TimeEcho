## Introduction
GregTech-TimeEcho is a modpack that bring some GT6 things to GTM.

## Contribute
GTTE uses [packwiz](https://packwiz.infra.link/ "packwiz") as development tool.

For Chinese language users which would like use Plain Craft Launcher for developing,
this [guide](http://docs.yw-games.top/posts/tutorial/modpack/packwiz.html "guide") may help you.
